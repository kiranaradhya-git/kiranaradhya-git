Here are some commonly used Unicode symbols you can use in Markdown tables and documentation:


| Symbol | Unicode | Description         |
|--------|---------|---------------------|
| ✔️     | U+2714  | Check mark          |
| ✖️     | U+2716  | Cross mark          |
| ➡️     | U+27A1  | Right arrow         |
| ★      | U+2605  | Black star          |
| ☆      | U+2606  | White star          |
| ⚠️     | U+26A0  | Warning             |
| ❗     | U+2757  | Exclamation mark    |
| 🔥     | U+1F525 | Fire                |
| 🟡     | U+1F7E1 | Yellow circle       |
| 🟢     | U+1F7E2 | Green circle        |
| 🔴     | U+1F534 | Red circle          |
| 📝     | U+1F4DD | Memo/Note           |
| 📌     | U+1F4CC | Pin                 |
| 📎     | U+1F4CE | Paperclip           |
| 🆕     | U+1F195 | New                 |
| 🛑     | U+1F6D1 | Stop sign           |
| 🏆     | U+1F3C6 | Trophy              |
| 🚩     | U+1F6A9 | Flag                |
| 🔑     | U+1F511 | Key                 |


You can copy and paste these symbols directly into your Markdown files or tables.  
For example:  
| Field | Status |  
|-------|--------|  
| Name  | ✔️     |
